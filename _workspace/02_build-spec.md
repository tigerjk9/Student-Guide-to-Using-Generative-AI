## 컴포넌트 데이터 스키마 (JS 배열 — 모두 `<script>` 상단 분리)
- `LIMITS[]` {nick, name, desc} ×5 → 아코디언
- `DANGERS[]` {n, title, sub, body, rule, color} ×5 → 아코디언
- `PROMPTS[]` {tag, color, title, body} ×6 → 복사형 스니펫
- `COMPARE[]` {title, color, good, goodEx, bad, badEx} ×4 → 비교카드
- `STOP[]` {k, word, ko, desc} ×4 → 플로우/리스트
- `SURVEY_Q[]` ×20, `SCALE[]`(1~4), `BANDS[]`(20-35/36-55/56-80) → 퀴즈
- `PLEDGE[]` ×10 → 체크리스트 (+ 5번 항목 사용시간 입력칸)
- `FLOW[]` 7단계, `PRINCIPLES[]` 3, `LITERACY[]`/`WHYIMP[]`/`ABOUT[]` 카드, `STOP_DEP[]`/`HALLU[]` 실천목록

## 인터랙티브 로직 명세
### 자가진단 퀴즈 (check)
- 상태 `ans={}` (문항index→점수). 라디오 `name="sq{i}"` 고유. `peer` 패턴(input 먼저).
- 합계 = Σ응답. 만점 80(20×4). 진행 표시 "응답 N/20". 미응답 있으면 결과 "모든 문항에 답해 주세요" 잠금.
- 20문항 모두 응답 시 결과 카드 노출: `BANDS.find(b=>s>=b.min&&s<=b.max)`. 밴드 색 배경 + 제목 + 본문 + (56+면 109/1388 강조).
- localStorage 키 `genai_survey` = {ans, done}. "다시하기" = 초기화.
- ※참고용 안내 문구 항상 노출.

### 약속 체크리스트 (pledge)
- localStorage 키 `genai_pledge` = {checks:{}, time:''}. 체크 토글 즉시 저장 + 진행률 바 + 체크 시 취소선.
- 5번 항목 아래 사용시간 텍스트 입력(`input` 이벤트로 time 저장, value 복원).
- "초기화" 버튼 → 키 삭제 + 재렌더.

### 복사형 스니펫 (smart)
- 복사 버튼 → `navigator.clipboard.writeText(PROMPTS[i].body)` + 토스트 "복사되었습니다 ✓" + 버튼 라벨 일시 변경. `.catch()` 무음 처리. `<pre>` 본문 `escapeHtml`.

### 아코디언/플로우/비교
- `<details>` + `.acc-arrow`. STOP은 정지표지(빨강 8각 느낌 원형 배지 S/T/O/P)로.

## 접근성·반응형 요구
- 큰 글자(본문 ≥14px, 제목 굵게), 터치 타깃 ≥44px, 모든 인터랙션 요소 `aria-label`/`type`, 라디오·체크 키보드 접근
